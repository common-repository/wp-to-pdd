Управляем ящиками на Яндекс Почта для домена прямо из админки Wordpress

== Description ==

С помощью этого плагина вы сможете **управлять почтовыми ящиками** в сервисе Ягднес Почта для домена прямо из админки своего сайта. Также вы можете включить или отключить автоматическое управление почтовыми ящиками, которое позволяет автоматически создавать, удалять и обновлять информацию при обновлении профиля пользователя в том числе и пароль, который всегда соответствует паролю на вашем сайте, если он не был изменен на почтовом ящике отдельно. [На этой странице](http://www.dampi.ru/plagin-wordpress-wp-to-pdd-versiya-0-9-1) можно получить исчерпывающую информацию о данном плагине.

== Installation ==

1. Качаем плагин
2. Распаковываем архив
3. Папку "wp-to-pdd" копируем в папку с плагинами вашего сайта "wp-content/plugins/"
4. Активируем плагин на странице "Плагины" в административной панели вашего сайта
5. Топаем на страницу "Почтовые ящики" (слева внизу) и действуем строго по приведенной, на этой странице, инструкции
